# SMK Rework Splitted Version for Guest

## Build Setup
**[Node.js](https://nodejs.org/en/download/current) >= 14.7.0**

```bash
# duplicate environment
$ cp .env.example .env

# setup the environment
# MIX_APP_URL = API URL

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
